//COM Port Setting

/* * *Attach the XBee module using the adapter to the computer *Open Control Panel *Open Hardware and Sound *Under Devices and Printers open Device Manager *Now expand Ports(COM and LPT) *Identify the COM port corresponding to the XBee and open its properties(Right-Click) *Click on the Port Settings tab. *Click on the Advanced button. A second dialog appears. *On this new dialog, there is “COM Port Number” option near the top. Change it to COM '9' and click on OK to close the dialog. You might get a warning message about other programs using the device and whether you wish to continue. Click on Yes. *Click on OK again to close the first dialog. *Close the Device Manager. Note that the Device Manager might still show the old COM port number. If you close it and re-open it, you will then be able to see the new COM port number listed. *Now the COM port has been set

  • */

//GUI screen instructions

/* * *The GUI provides two modes namely Manual and Autonoomous *Manual Mode: The user can control the bot using click buttons that appear on the screen after clicking the manual mode button

  • The available functions are: FORWARD; BACK ; LEFT ; RIGHT ; STOP ;BUZZER-ON;OFF; GRIPPER - PICK;LEAVE
  • Two different speeds of the FIREBIRD V are also available.
  • Autonomous Mode: The FIREBIRD V will use the logic in the code loaded on it along with its sensors to detect objects of the required colour.
  • First the colour sensor requires configuration.
  • So a black paper must be placed in front of the bot and then NEXT must be clicked
  • Now the colour of objects the bot must detect and sort must be picked
  • The bot turns left and searches for objects. So the bot must be placed such that the objects to be sorted are to its left.
  • After placing it NEXT must be clicked
  • The bot will now sort.
  • */
